Better Snowball Fight Ⅱ
更好的打雪仗2

~~打雪仗×~~ 黑帮火拼√

这是一个非常适合多人游戏的mod，添加了各种各样的雪球，工具，装备和武器。既可以互相无伤害拿雪球互喷，也可以互相竞技pvp，还可以造雪人跟雪人干仗或者让雪人去攻击其他生物


作者：Linngdu664，zx1316
